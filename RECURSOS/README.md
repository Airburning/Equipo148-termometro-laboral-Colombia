# RECURSOS

Carpeta para el material visual del proyecto (presentación y portada).
Todavía no se ha generado ninguno de estos archivos en este repositorio;
cuando existan, van aquí:

- `Presentacion.pptx` — PowerPoint original.
- `presentacion.pdf` — la misma presentación, exportada a PDF.
- `portada.png` — imagen de la diapositiva principal.

Mientras tanto, el material equivalente ya generado por el pipeline está
en `reports/` (`reporte_final.html` y `reports/figures/*.png`).
